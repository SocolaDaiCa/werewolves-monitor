## ğŸ“ Key Concepts

### Night Order (1-42)
Each role acts in a specific order during night phase

### Action Types (6 Total)
- **SELECT_PLAYER**: Choose 1 person
- **DUAL_SELECT**: Choose 2 people
- **TEXT_INPUT**: Write message
- **DUAL_OPTION**: Choose A or B
- **ACKNOWLEDGE**: Just confirm
- **NONE**: No action

### Nightly Types
- **ALWAYS**: Called every night
- **FIRST_NIGHT**: Called only on night 1
- **NEVER**: Not called at night (e.g., Hunter shoots on day)

### Limited Abilities
Some can be used once per game (e.g., Witch potions)

### Relationships
Players can be linked (lovers, possessed, cursed)
